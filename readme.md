### Emphasis {#emphasis}
-----

``** __``     bold text       `**bold** __bold too__`<br>
``* _``       italic text     `*italic* _italic too_`<br>
``~~``        strikethrough   `~~strikethrough~~`<br>
``combine!``  combine em!     `**_italicbold_** ~~_italicstrikethrough_~~`<br>
``~ ~``       subscript       `CO~2~`<br>
``^ ^``       superscript     `e=mc^2^`<br>
``==``        highlight       `==highlight==!`<br>
